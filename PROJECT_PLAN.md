# 프로젝트 설계서: 나만의 시 → 노래 앨범 생성 서비스

## 1. 서비스 개요
- 사용자가 직접 쓴 시를 입력하면, AI가 발라드/엔카풍 곡을 자동 생성하고 보컬(TTS) 합성까지 입혀 하나의 개인 앨범 트랙으로 완성하는 웹 서비스
- 목적: “나만의 시가 곡으로 탄생하는 경험” 제공

## 2. 주요 기능
1. **시 입력 및 곡 생성**
   - 사용자가 시를 입력하고, 곡 스타일(발라드/엔카) 선택
   - AI API를 통해 가사 리라이팅, 작곡, TTS/보컬 합성
   - 완성된 곡(mp3) 다운로드 및 앨범에 저장
2. **앨범 관리**
   - 사용자가 만든 곡 리스트(앨범) 조회 및 감상
   - 곡별 재생, 삭제, 정보 확인
3. **회원 관리(추후)**
   - 이메일/소셜 로그인, 내 앨범 관리
4. **과금/결제(추후)**
   - 1곡 무료, 이후 곡당 과금/패키지 판매

## 3. 기술 스택
- **Frontend**: React + Vite + Tailwind CSS
- **Backend**: Firebase Functions (AI API 연동)
- **DB/Storage**: Firebase Firestore, Firebase Storage

## 4. DB 스키마 (Firestore)
- Users: id, email, name
- Poems: id, text, userId
- Songs: id, poemId, style, audioURL

## 5. 화면 설계
- [메인] 시 입력/곡 생성 폼
- [앨범] 곡 리스트 및 플레이어
- [로그인/회원정보] (추후)

## 6. 개발 단계
1. 프로젝트 초기 세팅 (완료)
2. 시 입력/곡 생성 UI (완료)
3. 앨범(곡 리스트) UI 및 플레이어
4. Firebase 연동 및 AI API 연동
5. 회원/결제 기능 (추후)

## 7. 기타
- Vercel 배포, PWA/TWA 지원 예정
- AI API/보컬 합성은 외부 서비스 활용
